# Install

1. tmux:


```
brew install tmux
```

2. tpm:


```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

3. bash:

```
brew install bash
```

4. ruby:


```
brew install ruby
```

5. reattach-to-user-namespace:


```
brew install reattach-to-user-namespace
```

6. urlview:

```
brew install urlview
```
